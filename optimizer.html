BIMData.io Optimizer is an IFC file optimizer. This tool is capable of reduicing the size of IFC files in a few minutes and allows the user to view the 3D model on any browsers.

Back →

BIMData's 3D viewer is an online tool that allows architects, and professionals working in the building industry, to work together on 3D models.

It allows them to navigate on their models and analyze and comment it collectively.

This is a complex tool that needs some guidance, that's why we created an onboarding.
