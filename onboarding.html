BIMData's 3D viewer is an online tool that allows architects, and professionals working in the building industry, to work together on 3D models.

← Back

BIMData.io viewer allows users to navigate on their models and analyze and comment it collectively.

This is a complex tool that needs some guidance, that's why we created an onboarding.

I created an onboarding step by step to help the user at the beginning of their journey with the software.

I used the same design system BIMData.io was already using in their interface and implemented some new designs and illutrations.

Illustrations

I created isometric illustrations for every section card to compartimente the different sections of the tutorial. I used a range of blue pastel colors to create the 3G effect on the buildings.

Card style

Every step has the shape of a card placed next to the 3D model itself so that users can interact with a 3D model with the onboarding.

There are 3 types of cards:

The step cards (on the left): They indicate the main steps that the user need to go through during the onboarding process. They include an illustration element, a title, a small description and navigation buttons.

The basic action cards (above): They show to the users simple actions to navigate on the model, like how to make the model rotate or how to zoom in, zoom out, select an element and so on. The image will be an animated gif.

The multi-actions cards (above): Some interactions with the model need several actions, like how to hide or display an element for example. These actions are explained on these types of cards with also a gif animated image.
